---

## 项目速览

- 主控芯片: STC15F2K61S2
- 开发语言: C
- 主要工具: Keil + STC-ISP
- 当前主线工程: `16届蓝桥杯真题/`

核心模块覆盖:

- DS18B20 温度采集
- 超声波测距（superwave）
- AD/DA 采样与输出
- 数码管显示
- 按键输入
- 串口通信

## 快速开始

1. 用 Keil 打开目标工程 `project.uvproj`
2. 确认芯片与时钟参数
3. 全编译，检查 `Objects/` 与报错信息
4. 使用 STC-ISP 下载并上板验证

---

## 训练建议（精简版）

- 每次只改一个工程，避免跨工程混改
- 先单模块验证，再做整题联调
- 对阈值、显示格式、时序窗口改动做记录
- 固定保留一个“可过评测”的稳定版本

记录模板:

```text
日期:
所属题目:
改动内容:
改动原因:
影响模块:
回归结果:
后续计划:
```
